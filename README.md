# CRUD Java JDBC + MariaDB

Projeto de estudo desenvolvido durante o curso Jovem Programador.

## O que o projeto faz
CRUD completo de Clientes e Pedidos com conexão a banco MariaDB.

## Tecnologias
- Java
- JDBC
- MariaDB

## Como executar
1. Crie um banco MariaDB chamado `ESTUDO`
2. Configure as credenciais em `ConexaoMdb.java`
3. Execute `Main.java`
